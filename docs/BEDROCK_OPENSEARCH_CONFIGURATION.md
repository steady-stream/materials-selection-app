## How to Recreate

### Step 1: Create OpenSearch Serverless Collection

```bash
# Create collection (name will be auto-generated by Bedrock)
aws opensearchserverless create-collection \
  --name "bedrock-knowledge-base-$(date +%s)" \
  --type VECTORSEARCH \
  --description "Bedrock Knowledge Base vector storage" \
  --region us-east-1
```

**Or use AWS Console:**

1. Navigate to OpenSearch Service → Serverless collections
2. Create collection
3. Name: Use any name with `bedrock-knowledge-base-` prefix
4. Type: Vector search
5. Standby replicas: Disabled (default)
6. Encryption: AWS owned key (auto)

**Wait 5-10 minutes for collection to become ACTIVE**

### Step 2: Create Bedrock Knowledge Base

**Option A: AWS Console (Recommended for first-time)**

1. Navigate to Amazon Bedrock → Knowledge Bases
2. Click "Create knowledge base"
3. Knowledge base details:
   - Name: `MaterialsSelectionKB`
   - Description: `Construction project specifications and documents`
   - IAM Role: Use existing role `AmazonBedrockExecutionRoleForKnowledgeBase_ax0k4`
4. Configure data source (optional - can skip for now):
   - Skip if not uploading documents yet
5. Select embeddings model:
   - Model: `Titan Embeddings G1 - Text`
   - Full ARN: `arn:aws:bedrock:us-east-1::foundation-model/amazon.titan-embed-text-v1`
6. Vector database:
   - Select: OpenSearch Serverless
   - Choose your newly created collection
   - Vector index: `bedrock-knowledge-base-default-index` (auto-created)
   - Vector field: `bedrock-knowledge-base-default-vector`
   - Text field: `AMAZON_BEDROCK_TEXT`
   - Metadata field: `AMAZON_BEDROCK_METADATA`
7. Review and create

**Option B: AWS CLI**

```bash
# Get the collection ARN from Step 1
COLLECTION_ARN="arn:aws:aoss:us-east-1:634752426026:collection/YOUR_COLLECTION_ID"

# Create Knowledge Base
aws bedrock-agent create-knowledge-base \
  --name "MaterialsSelectionKB" \
  --description "Construction project specifications and documents" \
  --role-arn "arn:aws:iam::634752426026:role/service-role/AmazonBedrockExecutionRoleForKnowledgeBase_ax0k4" \
  --knowledge-base-configuration '{
    "type": "VECTOR",
    "vectorKnowledgeBaseConfiguration": {
      "embeddingModelArn": "arn:aws:bedrock:us-east-1::foundation-model/amazon.titan-embed-text-v1"
    }
  }' \
  --storage-configuration '{
    "type": "OPENSEARCH_SERVERLESS",
    "opensearchServerlessConfiguration": {
      "collectionArn": "'"$COLLECTION_ARN"'",
      "vectorIndexName": "bedrock-knowledge-base-default-index",
      "fieldMapping": {
        "vectorField": "bedrock-knowledge-base-default-vector",
        "textField": "AMAZON_BEDROCK_TEXT",
        "metadataField": "AMAZON_BEDROCK_METADATA"
      }
    }
  }' \
  --region us-east-1
```

### Step 3: Verify Setup

```bash
# List knowledge bases
aws bedrock-agent list-knowledge-bases --region us-east-1

# Check status
aws bedrock-agent get-knowledge-base --knowledge-base-id YOUR_NEW_KB_ID --region us-east-1
```

**Expected Status:** ACTIVE (may take 2-3 minutes)
